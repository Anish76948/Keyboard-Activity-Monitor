# Keyboard Activity Monitor

This project was created to understand how keyboard input can be captured and transmitted using Python.

## Features
- Captures keyboard input
- Sends data using socket programming
- Displays and stores input on the server side

## Tech Used
- Python
- Socket Programming
- pynput

## What I learned
- Handling keyboard events in Python
- Basic networking using sockets
- Client-server communication

## Note
This project is developed strictly for educational purposes to understand how input handling and communication works in systems.
